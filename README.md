# Registration form
A GUI registration form

A modern user registration form built with Python and Tkinter. The application features a clean and responsive user interface with input validation, password visibility toggle, hover effects, and real-time user interaction.

Features
User-friendly graphical interface
Name, Email, Phone, and Password fields
Email format validation
Phone number validation
Password length verification
Show/Hide password functionality
Modern dark-themed design
Interactive hover effects on buttons
Error handling and validation messages
Responsive layout with custom styling

Technologies Used
Python
Tkinter
Regular Expressions (Regex)
Purpose

This project was developed to practice GUI development in Python, form validation techniques, and modern user interface design principles.
